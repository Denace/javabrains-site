About the site

Java Brains is an Enterprise Java tutorial website developed and maintained by me, Koushik Kothagal, as a free time project. This is a non-commercial website; it is not intended to, and it does not, make money. This means two things: Firstly, all tutorials are free. Secondly, there are no ads. You'll see no ads on the site. And no ads on Youtube videos. I hope you'll appreciate the better user experience that results from it!

Technologies used

This site is built using Jekyll and AngularJS. It is hosted on Github pages. The code for the entire site is available on Github here.

I am not a UI developer by any stretch, and the frameworks that greatly helped me put together this site are Bootstrap and the Goldilocks approach. There are plenty to things that can be fixed / enhanced. Feel free to send me a pull request with any patches, UI enhancements or video transcripts / summaries. I would be happy to credit you for them. If you'd like to help, but don't know how, reach out to me on LinkedIn.
